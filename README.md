# Forging Destiny

Forging Destiny is a top-down 2D action game built in Godot 4.

## Status
Early development. Core systems such as player movement, combat, enemies, and weapons are actively being built and iterated on.

## Tech
- Godot 4.x
- GDScript

## Notes
This repository is public for development tracking and learning purposes. Expect unfinished systems and frequent changes.
